# Gemini Base Reference Prompt Pack

Use these prompts to generate one male and one female base reference for each animal.

Goal:
- Gemini handles the character design.
- Local tooling handles color variants and final game-ready sprite prep.
- Prompts are self-contained and should not assume any prior knowledge of the project name.

Recommended save names:
- `rat-base-reference.png`
- `crow-base-reference.png`
- `fox-base-reference.png`
- `snake-base-reference.png`
- `deer-base-reference.png`
- `frog-base-reference.png`
- `pigeon-base-reference.png`
- `raccoon-base-reference.png`
- `squirrel-base-reference.png`
- `goose-base-reference.png`

## Order

1. `rat`
2. `crow`
3. `fox`
4. `snake`
5. `deer`
6. `frog`
7. `pigeon`
8. `raccoon`

## 1. Rat

```text
Generate an IMAGE ONLY.

Do not answer with text.
Do not describe limitations.
Do not provide configuration data.
Do not provide JSON.
Do not provide a table.
Do not provide a written explanation.

Create exactly one clean reference image for a 2D desktop virtual pet game showing:
- one male rat
- one female rat

This prompt is self-contained. Do not rely on previous images.

Reference image rules:
- transparent background preferred
- no poster layout
- no animation board
- no labels
- no text
- no decorative frame
- only two characters in the image
- both characters should face the same side
- both characters should be fully visible
- keep both characters separated and easy to read
- place male on the left, female on the right

Style:
- pixel art
- crisp edges
- slightly moody, cute, vulnerable desktop-pet tone
- compact body
- rounded ears
- whisker-forward face
- thin tail
- scrappy but cute
- male should read slightly larger and bolder
- female should read slightly smaller and calmer

Purpose:
- this is a base design reference for later sprite production in a small desktop virtual pet game
- keep the silhouettes clean and readable
- prioritize consistent side-view readability over dramatic posing

Return only the generated image.
```
