# Gemini Age Reference Prompt Pack

Use these prompts to generate `baby` and `teen` reference images for each animal.

Goal:
- Gemini handles age-specific character design reference art.
- Local tooling later derives color variants and game-ready sprite prep.
- Prompts are self-contained and do not assume prior knowledge of the project name.

## 1. Rat / Baby

```text
Generate an IMAGE ONLY.

Do not answer with text.
Do not describe limitations.
Do not provide configuration data.
Do not provide JSON.
Do not provide a table.
Do not provide a written explanation.

Create exactly one clean reference image for a 2D desktop virtual pet game showing:
- one baby rat

This prompt is self-contained. Do not rely on previous images.

Reference image rules:
- transparent background preferred
- no poster layout
- no animation board
- no labels
- no text
- no decorative frame
- only one character in the image
- character should face to the side
- character should be fully visible
- keep the silhouette clear and easy to read

Style:
- pixel art
- crisp edges
- slightly moody, cute, vulnerable desktop-pet tone
- compact rat body
- rounded ears
- whisker-forward face
- thin tail
- baby should read as very small, vulnerable, rounder, and softer than the adult version
- oversized head and slightly shorter limbs are acceptable if still readable

Purpose:
- this is a base design reference for later sprite production in a small desktop virtual pet game
- keep the silhouette clean and readable
- prioritize a consistent side-view pose over dramatic posing

Return only the generated image.
```
