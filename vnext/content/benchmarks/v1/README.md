# Benchmark Suite v1

Approved benchmark cases live in `approved/` and are treated as immutable once committed.
Draft cases belong in `draft/` until a later curation phase promotes them.

The v1 axes are:

- `chat`
- `tool-use`
- `retrieval`
- `safety`
- `perf`

An empty `approved/` folder is not a pass. The suite reports `NoBaseline` until reviewed cases are present.
